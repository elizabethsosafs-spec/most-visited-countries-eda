# 🌍 Most Visited Countries in the World (2021)

## 📊 Project Overview
This project analyzes the 50 most visited countries in the world based on international tourist arrivals.

The goal is to identify global tourism patterns and understand which countries attract the highest number of international visitors.

---

## 📁 Dataset
- Source: Kaggle
- Countries: 50
- Variables: Country, Tourist Arrivals
- Year: 2021 (based on dataset description)

---

## 🧹 Data Notes
Two datasets were provided (ranking and arrivals).  
An inconsistency was found between them, so the analysis focuses on the tourist arrivals dataset.

---

## 📈 Key Insights
- France is the most visited country in the dataset.
- Spain and the United States follow in second and third place.
- Tourism is concentrated in a small number of countries.
